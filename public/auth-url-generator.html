3. 設置步驟

在微信公眾平台設置自定義菜單：

  1. 登錄微信公眾平台
  2. 進入 "自定義菜單"
  3. 點擊 "添加菜單"
  4. 菜單名稱：輸入您想要的名稱（如 "獲取OpenID"）
  5. 菜單類型：選擇 "跳轉網頁"
  6. 菜單鏈接：貼上上面生成的授權 URL
  7. 點擊 "保存並發布"

4. 測試流程

完整的授權流程：

  1. 用戶在微信中點擊自定義菜單
  2. 跳轉到微信授權頁面
  3. 用戶點擊 "確認登錄"
  4. 微信將用戶重定向到您的 callback 頁面，並帶上 code 參數
  5. 您的網站使用 code 獲取用戶的 OpenID 和基本信息

5. 重要提醒

  • 必須在手機微信內測試，不要用電腦瀏覽器直接打開授權 URL
  • 確保網頁授權域名已配置：openid.meditechmacao.com
  • 確保公眾號是服務號且已認證
  • callback 頁面必須能正常訪問
