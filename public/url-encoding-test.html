URL 編碼測試工具

1. 原始 URL 分析

您的跳轉地址：

https://openid.meditechmacao.com/wechat-auth

需要編碼的字符：

  • 冒號 (:) → %3A
  • 斜杠 (/) → %2F
  • 點 (.) → %2E
  • 連字符 (-) → %2D

4. 手動編碼驗證

手動編碼步驟：

  1. https:// → https%3A%2F%2F
  2. openid.meditechmacao.com → openid.meditechmacao.com（域名部分通常不需要編碼）
  3. /wechat-auth → %2Fwechat-auth

完整編碼結果：

https%3A%2F%2Fopenid.meditechmacao.com%2Fwechat-auth
