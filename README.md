# 微信公眾平台 OpenID 集成方案

## 項目概述
這個項目演示如何通過微信公眾平台自定義菜單獲取用戶OpenID，並在網站中使用OpenID獲取微信用戶資料。

## 功能特性
- 微信公眾平台自定義菜單配置
- 網頁授權獲取OpenID
- 獲取微信用戶基本信息
- 完整的測試和部署方案

## 技術棧
- 後端：Node.js + Express
- 前端：HTML + JavaScript
- 微信API：微信公眾平台API

## 目錄結構
```
openid-new/
├── server/           # 後端服務器
├── public/           # 前端靜態文件
├── config/           # 配置文件
├── docs/             # 文檔
└── README.md         # 項目說明
```
